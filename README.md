# Smart AC Control System

## 📌 Description
This project is a smart air conditioner control system developed using Java GUI and Arduino.  
The system allows users to control the AC remotely via a graphical interface.

## ⚙️ Technologies Used
- Java (Swing GUI)
- Arduino
- Serial Communication (USB)

## 🔧 Features
- AC ON → LED turns ON
- AC OFF → LED turns OFF
- Fan Mode → LED blinks
- Reset → System resets (LED OFF)

## 💡 How it works
The Java application sends commands to Arduino via serial communication.  
Arduino reads these commands and controls the LED accordingly.

## 🎯 Purpose
This project simulates a smart home air conditioning system.
